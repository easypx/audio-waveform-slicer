# 🎵 Audio data preparation for Sonauto uploads

**1. Audio Slicer**: Extract specific sections and export them concatenated or individually.

**2. Concat Audio**: Loads multiple audio files, can change their order, and saves the concatenated audio in various formats. (Ogg is saved 1:1 via streaming from the media recorder and is slow. Faster routine is planned.)
